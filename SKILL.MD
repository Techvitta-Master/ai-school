
# AI School MVP

## Goal
Build a simple school evaluation system for one school.

## Roles
- School
- Teacher
- Student

## Features
- Create classes (6–10)
- Add teachers (subject-wise)
- Add students (with roll number)
- Assign students to class
- Assign teacher to class
- Teacher uploads answer sheet using roll number
- Generate dummy evaluation
- Student sees result

## Rules
- Keep UI simple
- No extra features
- Use Supabase for database
- Use dummy AI only
